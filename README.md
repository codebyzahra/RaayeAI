# 🛍️ RaayeAI
**Roman Urdu Product Review Sentiment Analyzer**

🔗 **Live Demo:** https://raayeai-sentiment1.streamlit.app/

## About
NLP-based sentiment analysis system for code-mixed Roman Urdu product reviews scraped from Daraz.pk

## Features
- Live review analysis
- Confidence score
- Seller suggestions
- Trained on 16,000+ real Daraz reviews

## Tech Stack
Python | Scikit-learn | TF-IDF | Streamlit | NLP

## Model Performance
- Negative: 93.6% accuracy
- Neutral: 92.6% accuracy  
- Positive: 90.7% accuracy
